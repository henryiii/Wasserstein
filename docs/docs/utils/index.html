Utils

Python

check_emd_status

wasserstein.check_emd_status(status)

Examines the return status of an EMD computation and raises an appropriate RuntimeError if it is not 0.

Arguments

  • status : int
    • The status of the EMD computation. Zero means success and non-zero values indicate some sort of failure.

phi_fix

wasserstein.phi_fix(phi, ref_phi)

Ensures that phi is within \pi of ref_phi.

Arguments

  • phi : float
    • An azimuthal angle.
  • ref_phi : float
    • A reference azimuthal angle that should be numerically within 2\pi of phi, which is always satisfied if both phi and ref_phi are in a standard range like [0,2\pi).
